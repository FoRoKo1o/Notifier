Notifier App

About app

Simple reminder, you can add notification to list and whenever desired date came yours Windows PC will give you notification with given message.

SECURITY

THIS APP IS NOT 100% SECURE!!!
Data base is encrypted but everyone can acces it and try to breach it.

Important

API is hosted on a different Replit project and sometimes requires posting login/register form twice. Once to wake up the API project (takes around 10 seconds) and a second one to perform the login or register action.

Source code

App itself is hosted on Replit, so you can view code from there but also you can find source code here: Github(in progress).
